Daily: August 14th, 2013

Version 1.1.3

Valerio Mazzeo August 14, 2013 Updates No Comments
  • Fixed a problem that was preventing the source code browser to show .py files correctly.
  • Fixed a problem that was preventing the source code browser to correctly display the content of subfolders.
  • Fixed a problem that was preventing the source code browser to correctly show commits on a default branch which name was not “master”.
Note – If you are still experiencing problems in the source code browser after upgrading, it is possible that the app is still holding locally some old cached contents. In order to resolve this, please unlink the app from your current host and login in again.
